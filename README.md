# Mustache Core Library for Android

[![](https://jitpack.io/v/mustachedk/core-lib-an.svg)](https://jitpack.io/#mustachedk/core-lib-an)

###### Add this to your root build.gradle
```bash
allprojects {
  repositories {
    maven { url 'https://jitpack.io' }
  }
}
```

###### Add this to your root build.gradle
```bash
dependencies {
  implementation 'com.github.mustachedk:core-lib-an:0.2.25'
}
```

#### Currently Contains:
##### Utilities
* TaskHandler
* NavigationBar
* InternetConnection
* LocationUtil
* NetworkUtil
* PermissionUtil
* PixelUtil
* ScreenUtil
* ValidateUtil
* ViewUtil

##### Views
* OneClickButton

##### Dialogs
* BottomSheetMenu

##### Base Classes:
* DataBindingAdapter (DataBindingViewHolder)
* SectionHeaderListAdapter
* HeaderListViewPagerFragment
